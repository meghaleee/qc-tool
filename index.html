// Settings — imgbb API Key

When you export, screenshots are uploaded to imgbb (free image hosting) and the Excel file gets a real clickable link in the Screenshot column — no extra files needed.

How to get your free API key
1
Go to imgbb.com/signup and create a free account.
No credit card required. Takes 30 seconds.
2
Once logged in, go to api.imgbb.com
This is the API page for your account.
3
Click Get API key — you'll see a string of random characters.
Looks like: a1b2c3d4e5f6...
4
Copy the key and paste it in the field below. Click Save Key.
Saved in your browser — you only need to do this once.
✓
Now when you click ↓ Export, screenshots upload automatically and your Excel gets a live https:// link per issue.
